| HIGH | `apps/backend/src/http/fs.routes.ts:16`, `apps/backend/src/http/fs.routes.ts:30` | `/fs/validate` and `/fs/list` pass caller-supplied paths directly to `fs.stat`/`fs.readdir`. Relative paths also resolve against the backend process cwd. | Canonicalize path input once. Require absolute local paths, reject UNC/device roots unless explicitly enabled, normalize/resolve before use, and return the normalized path. |
